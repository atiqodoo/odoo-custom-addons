POS: Block Selling Below Cost Price

Protect your margins by preventing sales below cost.

Key Features

Block Below-Cost Sales: The module performs a final check on the payment screen to ensure no product is sold for less than its cost.

Automatic Price Correction: If an issue is found, prices are automatically adjusted to the cost price.

Clean & Professional: Built according to Odoo 18 best practices for stability and future compatibility.
