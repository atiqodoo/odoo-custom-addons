POS: Block Price Below Cost
=============================

This module enhances the Odoo Point of Sale by preventing users from selling products for less than their defined cost price.

.. image:: static/description/icon.png
   :width: 128px
   :height: 128px

Key Features
============

*   **Price Protection**: Automatically blocks the validation of any order containing products priced below their cost.
*   **Auto-Correction**: When a below-cost item is detected at the payment screen, its price is automatically reset to the cost price and the discount is removed.
*   **Clear Notifications**: The user is presented with a clear and detailed alert/popup explaining which products were corrected, ensuring transparency.
*   **Universal Compatibility**: Works on Odoo 18 Community, Enterprise, and Odoo.sh (SaaS).

Configuration
=============

No configuration needed.

Installation
============

Standard Odoo module installation:

1.  Download and place the module in your Odoo addons directory.
2.  Restart the Odoo server.
3.  Navigate to **Apps**, click **Update Apps List**.
4.  Search for "POS: Block Price Below Cost" and click **Install**.
