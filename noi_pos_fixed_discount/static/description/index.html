POS Fixed Discount

Odoo 18.0

POS Fixed Discount enhances the Point of Sale experience by allowing cashiers to apply a fixed amount discount on POS order lines, in addition to the default percentage-based discount.


✨ Key Features

  • Apply fixed amount discount per POS order line
  • Switch between Fixed and Percentage discount modes
  • Percentage discount remains the default behavior
  • Discount details visible directly on POS order lines
  • Discount information shown on POS Order form


🛒 POS Discount Flow

1️⃣ Choose Discount Type on POS Order Line

While adding products in POS, the cashier can choose whether the discount should be applied as a Fixed Amount or a Percentage.

Discount type selector on POS order line


2️⃣ Discount Applied on Order Line

After selecting the discount type and value, the discount is immediately reflected on the POS order line.

Fixed discount applied on POS order line
💡 Note: Fixed discount applies only to the current order line, while percentage discount follows the default POS behavior.


📄 POS Order Form View

Once the order is confirmed, the applied discount (fixed or percentage) is clearly visible on the POS Order form for easy tracking and reporting.

Discount details on POS order form


⚙️ Technical Details

  • Compatible with Odoo 18.0
  • Depends on point_of_sale
  • No core modification
  • Clean and modular JS implementation
