POS Deferred / Pending Collection

Backend-Only Solution for Paint Retail Stores

Allow customers to pay the full bill in POS, take some items immediately, and leave the rest in the shop for later collection - all managed 100% from the backend with zero changes to the POS interface.

Key Features

🎯 100% Backend Operation

Zero changes to POS interface - no buttons, no checkboxes, no wizards during checkout

📦 Inventory Reserved

Items automatically moved to dedicated holding location - cannot be sold again

🔍 Easy Search

Find pending items by customer, phone, POS receipt, or barcode scan

🏷️ Printable Labels

Auto-generated labels with barcodes for physical items in storage

How It Works

Step 1: Normal POS Sale

Customer pays full amount → Receipt printed → Order closed (no extra action)

Step 2: Register Deferred Collection (30-60 seconds)

Backend user opens POS order → Clicks "Register Deferred Collection" → Adjusts quantities → System creates record + moves stock + prints label

Step 3: Physical Storage

Staff attaches label to items → Places on "Pending Collection" shelf

Step 4: Customer Returns

Staff searches pending collections → Opens record → Marks collected → Stock released
