Enhanced Stock Card Report

Complete stock movement report with AVCO costing, purchase tracking, and POS sales analysis.

Key Features

📊 Comprehensive Movement Tracking

  • Physical IN/OUT/Balance quantities
  • All move types: Purchase, Sales, POS, Manufacturing, Transfers, Adjustments
  • Internal transfer tracking with dual-line display
  • Chronological ordering (date + sequence + ID)

💰 Financial Integration

  • AVCO Unit Cost from stock.valuation.layer
  • Purchase Unit Price & Amount (incl. VAT)
  • POS Unit Price & Amount (incl. taxes)
  • Running totals for purchases and POS sales
  • Inventory value tracking

🎯 100% Reconciliation

  • Stock quantities match stock.quant
  • Values from stock.valuation.layer (accounting source of truth)
  • Purchase amounts match supplier invoices
  • POS amounts match session cash totals

📈 Advanced Analysis

  • Per-transaction margin analysis
  • Price variance tracking
  • Purchase vs selling price comparison
  • Opening balance calculation
  • Flexible date range filtering

📄 Multiple Export Formats

  • PDF Report - Professional layout with 16 columns
  • Excel Export - Full data with formatting and analysis
  • Separate page per product option
  • Summary sheet with totals

Usage

  1. Navigate to Inventory → Reporting → Stock Card Report
  2. Select products (or leave empty for all)
  3. Choose warehouse or specific locations
  4. Set date range
  5. Configure options (internal transfers, grouping)
  6. Click Print PDF or Export Excel

Technical Details

  • Version: 18.0.1.0.0
  • Odoo Edition: Enterprise (Community compatible)
  • Dependencies: stock, stock_account, purchase_stock, sale_stock, point_of_sale, mrp
  • License: LGPL-3

Requirements

  • Odoo 18.0 Enterprise or Community
  • Anglo-Saxon accounting mode
  • Automated inventory valuation
  • AVCO (Average Cost) costing method
  • Python library: xlsxwriter (for Excel export)
