🚀 How It Works

Simple 4-Step Process:

  1. Select Customer - Search and choose from your customer list
  2. Choose Operation - Add or reduce points
  3. Enter Amount - Specify points and see new balance instantly
  4. Apply - Confirm and create accounting entries automatically

📊 Accounting Example

Scenario: Customer redeems 100 loyalty points worth $50
Debit: Loyalty Points Liability $50.00 Credit: Loyalty Points Expense $50.00

Journal entries are created and posted automatically upon confirmation!
